# Standard: RMA Photography & Documentation

- Photograph every board **before** any power-on attempt.
- Capture clear close-ups of U6/U11, RAM slots, rear ports, and any bent pins.
- Include scale/reference object when possible.
- Save all photos in the correct 04-evidence/ subfolder (asus-pro/, msi/, slim-sas/, etc.).
- Include pattern summary when multiple identical units are affected.
