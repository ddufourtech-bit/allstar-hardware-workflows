# New Tower Build Completion Checklist

- [ ] Chassis unboxed and mounting bolts + I/O shield installed
- [ ] Motherboard prepped (CPU, RAM, heatsink) on bench
- [ ] USB 3.0 cable attached **before** mounting motherboard
- [ ] Motherboard mounted in chassis
- [ ] PSU installed and **all** cables plugged in before final mounting
- [ ] Water cooling pump/radiator installed (if applicable)
- [ ] MSI EZ Control Hub + Magnetic EZ Bridge installed and powered (if applicable)
- [ ] Dual GPUs / add-on cards installed and powered
- [ ] RAID 0 / storage configured (if applicable)
- [ ] All cables routed and dressed (small cables first, power last)
- [ ] System powered on, BIOS time/date updated, Memtest started
