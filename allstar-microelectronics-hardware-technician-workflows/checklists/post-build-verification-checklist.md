# Post-Build Verification Checklist

- [ ] System POSTs successfully with no errors/beeps
- [ ] All fans (including CHA_FAN1) spin at correct speeds
- [ ] IPMI/BMC accessible and sensors reporting correctly
- [ ] All drives detected in BIOS
- [ ] RAID array healthy (if configured)
- [ ] CPU/RAM stress tested (Memtest86+ minimum 72 hrs for LEGO)
- [ ] Temperatures and fan curves verified under load
- [ ] Serial number assigned and labeled on chassis + shipping box
- [ ] All evidence photos taken and stored in 04-evidence/
- [ ] System ready for shipment or client hand-off
