# Pattern: Shipping Damage – Multiple H12DSi-N6 Units

**Status:** Pattern Identified  
**Date:** February–March 2025

## Problem Summary
Recurring pattern across multiple H12DSi-N6 boards from the same shipment.

## Pattern Observed
- Consistent damage to U6/U11 chipset ICs
- Bent rear ports (DisplayPort, serial)
- Loose internal brackets and fan assemblies
- No-power / no-POST on arrival

## Root Cause
Single shipment packaged improperly (units upside-down / insufficient padding)

## Resolution
- Bulk RMA submitted with pattern documentation
- All affected boards replaced

## Lessons Learned
- When multiple identical boards show the same damage, treat as shipment-wide issue and include pattern summary in RMA
