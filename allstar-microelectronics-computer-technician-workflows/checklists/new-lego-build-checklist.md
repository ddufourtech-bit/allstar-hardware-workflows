# New LEGO Build Completion Checklist

**Use before moving system to KVM / Memtest**

- [ ] Chassis prepped (tops/bottoms, fans, SSD clips, white bar, legs/pads/bolts)
- [ ] Motherboard populated per Sales Order (CPU, RAM, SSDs in correct slots)
- [ ] Thermal paste applied and heatsink torqued in star pattern
- [ ] All fan cables connected to correct headers (CHA_FAN1 verified)
- [ ] Water pump (if present) connected to AIO/CPU_FAN header
- [ ] PSU cabled (24-pin + 2× CPU 8-pin; PCIe/SATA only if needed)
- [ ] IPMI card (if present) fully cabled and dressed
- [ ] All add-on cards (GPU, RAID, etc.) seated and secured
- [ ] Front-panel and USB 3.0 cables attached
- [ ] System moved to KVM station and powered on for initial POST check
